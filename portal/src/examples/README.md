# Examples Reference

These files are **not imported** by the app — they're copy-paste references for
common patterns you'll use when building pages.

| File | Pattern |
|------|---------|
| `app-services.js` | Using `app.rest`, `app.toast`, `app.modal`, `app.events` |
| `table-page.js` | TablePage with columns, filters, forms, and VIEW_CLASS |
| `detail-view.js` | Detail View with header, tabs, DataView, actions |
| `form-page.js` | FormPage with auto-save and tabbed layout |
| `dashboard-page.js` | Dashboard with stats cards and metric charts |
| `model.js` | Model + Collection + forms + helpers |
| `templates.md` | Mustache template cheat sheet |
